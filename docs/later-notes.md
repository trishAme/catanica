# Later Notes

Short backlog of ideas to revisit after the first playable prototype feels stable.

## Gameplay Ideas

- Add table lamps as real level objects.
  - Lamps should cast warm local light.
  - The cat can knock a lamp down.
  - When a lamp falls, its light disappears.
  - This can make the room darker without adding a timer.
- Add small harmless household props that are visually clear: books, blanket, mug, laptop, cat toy.
- Add multiple sleep spots across levels: cat bed, shelf blanket, cat hammock, laptop keyboard.

## Art Ideas

- Replace procedural placeholder sprites with a consistent cozy pixel-art asset set.
- Create a real cat spritesheet with idle, walk, jump, sniff, eat, knock, angry, and sleep poses instead of drawing the cat only with code.
- Consider using AI-generated bitmap drafts as references, then clean them into a small controlled pixel-art spritesheet.
- Add richer day-to-night palettes: daylight, golden evening, dusk, night.
- Add window clouds and stars only after enough purrs are collected.
- Make plant silhouettes more recognizable before adding detailed plant cards.

## Education Ideas

- Add a small plant encyclopedia screen unlocked by sniffing.
- Keep sniff cards neutral; reveal safety category only after the player acts.
- Add source links in a plant detail view or README appendix.

## Accessibility Ideas

- Add remappable controls.
- Add a reduced-flash mode for failure state effects.
- Add readable text scale options for plant cards.
