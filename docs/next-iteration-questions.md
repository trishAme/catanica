# Next Iteration Questions and Ideas

## Questions

- How explicit should the learning layer be: cozy facts only, or a stronger warning flow for dangerous plants?
- Do we want a plant journal/collection screen where sniffed plants are saved between rooms?
- Should levels have a timer later, or should the first demo stay slow and exploratory?
- Which references are safe and fun enough for the tone: Azura-only, broader TES jokes, or subtle geek-room props?
- Should the cat have small personality traits by coat, or should coat remain purely cosmetic?

## Ideas

- Add desk lamps as knockable light sources; when the cat drops one, that lamp's light disappears.
- Add a “grandma gave me this plant” scenario pack with very common household plants.
- Add plant silhouettes that look intentionally similar, then teach the player to sniff and compare details.
- Add a tiny end-of-level recap: eaten, knocked, ignored, and one useful real-world reminder.
- Add animated cat idle states: tail flick, ear twitch, loaf, suspicious stare.
- Add a plant encyclopedia page in the README/demo so reviewers can see the curated data quickly.
